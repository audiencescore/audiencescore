# Score Spec

The versioned score math, licensed CC BY 4.0. The current version is
[score-spec-v0.1.md](score-spec-v0.1.md).

Two properties are constitutional (see [GOVERNANCE.md](../GOVERNANCE.md)):
the score function is public in every version, and manifests always carry
the version that produced them, so every historical score is reproducible.
Superseded versions stay in this directory forever.
