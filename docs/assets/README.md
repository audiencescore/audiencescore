# Brand assets

The project's wordmark and avatar. Intentionally minimal text marks; a
designed identity can replace them later without touching anything else.

| File | Size | Use |
|---|---|---|
| `avatar-AS.svg` / `.png` | 512×512 | Square org/repo avatar ("AS" monogram) |
| `wordmark-audiencescore.svg` / `.png` | 1280×320 | Horizontal wordmark for docs/site headers |

Palette: background `#0E1525` (deep navy), text `#F5F7FA` (near-white),
accent `#3BC97A` (verification green). SVG is the source of truth; the PNGs
are rendered from it (headless Chrome) — regenerate the PNG after any SVG
edit. These are intentionally minimal text wordmarks; a designed identity
can replace them later without touching anything else.
